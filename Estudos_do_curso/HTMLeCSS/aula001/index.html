O que é HTML?

HTML é uma linguagem de marcação de texto, ou seja, ele não é uma linguagem de programação.
Com o HTML, marcamos os elementos para informar ao navegador o que é o que.

O que é CSS?

CSS é uma linguagem de estilização. Com ela, damos "vida" ao nosso documento HTML.
Podemos mudar cores, fontes, tamanhos, posicionamento e muito mais.

Inserir imagens

O HTML 5 introduziu duas novas tags específicas para imagem: figure e figcaption. A tag figure define uma imagem em conjunto com a tag img. Além disso, permite adicionar uma legenda para a imagem por meio da tag figcaption.

Ex...

                
                    <figure>
                        <img src="img/matriz-musicdot.png" alt="Foto da matriz da musicdot">
                        <figcaption>Matriz da MusicDot </figcaption >
                    </figure>

Criando um link externo

Para criar um link externo, usamos a tag <a href="" target="_blank" rel="external" >texto do link</a>.

Neste caso, o link abrirá em uma nova aba, usamos os atributos target="_blank" e rel="external".

Exemplo com target e rel: Acessar o Google em nova aba

Para criar um link na mesma aba, usamos a tag <a href="">texto do link</a>, mas o atributo href apontará para um arquivo local.

Exemplo: Acessar o Google
