Alinhamento e Centralização

Formas mais comuns de centralizar elementos.

1. Horizontal (Elementos de Bloco)

Use margin: 0 auto; em um elemento que tenha uma width definida.

Eu sou um bloco centralizado (width: 70%). 
.elemento-bloco {
    width: 70%;
    margin: 0 auto; /* 0 para top/bottom, auto para left/right */
}

2. Horizontal (Texto e Itens Inline)

Use text-align: center; no elemento PAI.

Eu sou um texto centralizado. 
.container-pai {
    text-align: center;
}

3. Horizontal e Vertical (Flexbox)

Use display: flex;, justify-content: center; (horizontal) e align-items: center; (vertical) no PAI.

Flexbox!
.container-pai {
    display: flex;
    justify-content: center; /* Alinha no eixo X (principal) */
    align-items: center;   /* Alinha no eixo Y (cruzado) */
    height: 150px; /* Precisa de uma altura para centralizar verticalmente */
}

4. Horizontal e Vertical (Grid)

O método mais fácil. Use display: grid; e place-items: center; no PAI.

Grid!
.container-pai {
    display: grid;
    place-items: center; /* Atalho mágico para centralizar nos 2 eixos */
    height: 150px;
}

Layout com Flexbox (Passo a Passo)

Flexbox é um modelo de layout para organizar itens em uma única dimensão (linha ou coluna).

<!-- HTML Base -->
<div class="flex-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
</div>

1. Propriedades do Contêiner (PAI)

display: flex; (Obrigatório)

Ativa o Flexbox no contêiner pai.

1
2
3
.flex-container { display: flex; }

flex-direction: (Direção)

Define o eixo principal. O padrão é row (linha).

Exemplo: flex-direction: column;

1
2
3
.flex-container {
    display: flex;
    flex-direction: column; /* Outros: row, row-reverse, column-reverse */
}

justify-content: (Alinhamento no Eixo Principal)

Distribui os itens ao longo do eixo principal (horizontal se flex-direction: row).

Exemplo: justify-content: space-between;

1
2
3
.flex-container {
    display: flex;
    justify-content: space-between;
    /* Opções: flex-start, flex-end, center, space-between, space-around */
}

align-items: (Alinhamento no Eixo Cruzado)

Alinha os itens ao longo do eixo cruzado (vertical se flex-direction: row).

Exemplo: align-items: center;

1
2
3
.flex-container {
    display: flex;
    align-items: center;
    /* Opções: flex-start, flex-end, center, stretch (padrão), baseline */
}

flex-wrap: (Quebra de Linha)

Permite que os itens quebrem para a linha seguinte se não couberem.

Exemplo: flex-wrap: wrap;

1
2
3
.flex-container {
    display: flex;
    flex-wrap: wrap; /* Padrão é 'nowrap' */
}

2. Propriedades dos Itens (FILHOS)

flex-grow: (Crescer)

Define a capacidade de um item "crescer" para ocupar o espaço livre. O padrão é 0.

Exemplo: Item 2 tem flex-grow: 1;

1
2 (eu cresço)
3
.item-2 { flex-grow: 1; }

flex-shrink: (Encolher)

Define a capacidade de um item "encolher" se não houver espaço. O padrão é 1.

flex-basis: (Tamanho Base)

Define o tamanho inicial do item antes do espaço ser distribuído. (Ex: 200px, auto).

flex: (Atalho)

Atalho para flex-grow, flex-shrink e flex-basis.

/* flex: [grow] [shrink] [basis]; */
.item {
    flex: 0 1 auto; /* Padrão */
    flex: 1; /* Atalho para: 1 1 0% (cresce e encolhe, tamanho base 0) */
    flex: auto; /* Atalho para: 1 1 auto */
}

order: (Ordem)

Muda a ordem visual dos itens. O padrão é 0.

Exemplo: Item 1 tem order: 1;, Item 3 tem order: -1;

1 (order: 1)
2 (order: 0)
3 (order: -1)
.item-1 { order: 1; } /* Vai para o fim */
.item-3 { order: -1; } /* Vai para o início */

Layout com Grid (Passo a Passo)

Grid é um modelo de layout para organizar itens em duas dimensões (linhas E colunas).

<!-- HTML Base -->
<div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
</div>

1. Propriedades do Contêiner (PAI)

display: grid; (Obrigatório)

Ativa o Grid no contêiner pai.

grid-template-columns: e grid-template-rows:

Define o número e o tamanho das colunas e linhas. Esta é a propriedade mais importante.

Exemplo: grid-template-columns: 1fr 2fr 1fr; (3 colunas, a do meio é 2x maior)

1 (1fr)
2 (2fr)
3 (1fr)
4
5
6
.grid-container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* 1fr = 1 fração do espaço */
    grid-template-rows: 50px 100px; /* 1ª linha 50px, 2ª linha 100px */
    /* Outras unidades: 100px, 50%, auto */
}

gap: (Espaçamento)

Define o espaço ENTRE os itens do grid.

.grid-container {
    gap: 10px; /* Espaço de 10px nas linhas e colunas */
    
    row-gap: 20px; /* Só nas linhas */
    column-gap: 5px; /* Só nas colunas */
}

2. Propriedades dos Itens (FILHOS)

Permitem que um item ocupe mais de uma célula do grid.

grid-column: e grid-row: (Atalhos)

Define em qual linha/coluna o item começa e termina.

Exemplo: Item 1 ocupa 3 colunas, Item 3 ocupa 2 linhas, Item 5 começa na coluna 2 e ocupa 2 colunas.

1 (ocupa 3 colunas)
2
3 (ocupa 2 linhas)
4
5 (col: 2 / span 2)
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 colunas */
    gap: 10px;
}
.item-1 {
    grid-column: 1 / 4; /* Começa na linha 1, termina ANTES da 4 */
    /* Alternativa: grid-column: 1 / span 3; (Começa na 1, ocupa 3) */
}
.item-3 {
    grid-row: 2 / 4; /* Começa na linha 2, termina ANTES da 4 */
}
.item-5 {
    grid-column: 2 / span 2; /* Começa na linha 2, ocupa 2 */
}

3. Exemplo Prático: Layout de Página com grid-template-areas

Permite "desenhar" o layout nomeando as áreas.

Header
Main Content
Sidebar
Footer
.layout-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 colunas */
    grid-template-rows: auto 1fr auto; /* Header/Footer auto, Main 1fr */
    gap: 10px;
    grid-template-areas: 
        "header header header"  /* Linha 1: Header ocupa tudo */
        "main main sidebar"   /* Linha 2: Main ocupa 2, Sidebar 1 */
        "footer footer footer";  /* Linha 3: Footer ocupa tudo */
}
/* Agora, atribua as áreas aos itens */
.header  { grid-area: header; }
.main    { grid-area: main; }
.sidebar { grid-area: sidebar; }
.footer  { grid-area: footer; }

Variáveis CSS (Propriedades Customizadas)

Permitem armazenar valores (como cores, fontes, tamanhos) para reutilizar em seu CSS.

1. Declaração (Geralmente no :root)

Variáveis são declaradas com dois traços --.

:root {
    --cor-primaria: #007bff;
    --tamanho-fonte-padrao: 16px;
    --espacamento-medio: 15px;
}

2. Utilização (Com var())

Você usa a variável com a função var().

.botao {
    background-color: var(--cor-primaria);
    padding: var(--espacamento-medio);
}
p {
    font-size: var(--tamanho-fonte-padrao);
    color: var(--cor-primaria);
}

3. Exemplo Prático: Tema Light/Dark

Variáveis CSS brilham quando usadas para trocar temas. O JS apenas troca uma classe no <body>, e as variáveis fazem o resto.

Caixa de Exemplo

Este texto e fundo mudarão de cor.

Este link também. 
/* 1. Define as variáveis no :root (Modo Light) */
:root {
    --cor-bg: #ffffff;
    --cor-texto: #222222;
    --cor-link: #007bff;
}
/* 2. Redefine as variáveis quando o body tiver a classe .dark-theme */
body.dark-theme {
    --cor-bg: #222222;
    --cor-texto: #ffffff;
    --cor-link: #ffa500;
}
/* 3. Usa as variáveis */
body {
    background-color: var(--cor-bg);
    color: var(--cor-texto);
}
a {
    color: var(--cor-link);
}

// JavaScript para o botão
document.getElementById('theme-toggle-btn').addEventListener('click', () => {
    document.body.classList.toggle('dark-theme');
});

Tabelas HTML

Usadas para exibir dados tabulares (linhas e colunas).

Estrutura Básica

  • <table>: O contêiner da tabela.
  • <thead>: O cabeçalho da tabela.
  • <tbody>: O corpo da tabela.
  • <tfoot>: O rodapé da tabela (opcional).
  • <tr>: (Table Row) Define uma linha.
  • <th>: (Table Header) Célula de cabeçalho (negrito e centralizado por padrão).
  • <td>: (Table Data) Célula de dados.

Exemplo 1: Tabela Simples

Nome Matrícula Nota
Ana Silva 202401 9.5
Bruno Costa 202402 8.0 
<table class="tabela-exemplo">
    <thead>
        <tr>
            <th>Nome</th>
            <th>Matrícula</th>
            <th>Nota</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ana Silva</td>
            <td>202401</td>
            <td>9.5</td>
        </tr>
        <tr>
            <td>Bruno Costa</td>
            <td>202402</td>
            <td>8.0</td>
        </tr>
    </tbody>
</table>

Exemplo 2: Mesclagem de Células (colspan e rowspan)

  • colspan="2": Faz uma célula ocupar o espaço de 2 colunas.
  • rowspan="2": Faz uma célula ocupar o espaço de 2 linhas.
Aluno Provas
Prova 1 Prova 2
Maria 8.0 9.0
7.5 8.5
João 9.0 10.0 
<table class="tabela-exemplo">
    <thead>
        <tr>
            <th>Aluno</th>
            <th colspan="2">Provas</th>
        </tr>
        <tr>
            <th></th> <!-- Célula vazia para alinhar -->
            <th>Prova 1</th>
            <th>Prova 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td rowspan="2">Maria</td>
            <td>8.0</td>
            <td>9.0</td>
        </tr>
         <tr>
            <!-- Célula da Maria (rowspan) ainda está aqui -->
            <td>7.5</td>
            <td>8.5</td>
        </tr>
    </tbody>
</table>

Formulários HTML

Usados para coletar dados do usuário.

<!-- O <form> agrupa todos os campos -->
<form action="/url_para_onde_enviar" method="post">

    <!-- É fundamental usar <label> com 'for' (que aponta para o 'id' do input) -->
    <div class="form-group">
        <label for="nome">Nome (Texto):</label>
        <input type="text" id="nome" name="nome_usuario">
    </div>

    <!-- Dropdown (Select Box) -->
    <div class="form-group">
        <label for="curso">Curso:</label>
        <select id="curso" name="curso_usuario">
            <option value="">Selecione...</option>
            <option value="cc">Ciência da Computação</option>
        </select>
    </div>

    <!-- Botões de Rádio (Radio Buttons) - Só pode selecionar UM -->
    <!-- O atributo 'name' DEVE ser o mesmo para agrupar -->
    <div class="form-group">
        <label>Período:</label>
        <label><input type="radio" name="periodo" value="manha"> Manhã</label>
        <label><input type="radio" name="periodo" value="tarde"> Tarde</label>
    </div>

    <!-- Caixas de Seleção (Checkboxes) - Pode selecionar VÁRIOS -->
    <div class="form-group">
        <label>Interesses:</label>
        <label><input type="checkbox" name="interesses" value="frontend"> Front-end</label>
        <label><input type="checkbox" name="interesses" value="backend"> Back-end</label>
    </div>

    <!-- Área de Texto -->
    <div class="form-group">
        <label for="obs">Observações:</label>
        <textarea id="obs" name="observacoes" rows="4"></textarea>
    </div>

    <!-- Botões -->
    <div class="form-group">
        <button type="submit">Enviar</button>
        <button type="reset">Limpar</button>
    </div>

</form>

Galeria de Imagens (Responsiva com Grid)

Este método usa grid-template-columns com auto-fit e minmax para criar uma galeria perfeitamente responsiva sem precisar de Media Queries.

HTML

<div class="galeria-grid">
    <div class="galeria-item"><img src="..."></div>
    <div class="galeria-item"><img src="..."></div>
    <div class="galeria-item"><img src="..."></div>
</div>

CSS

.galeria-grid {
    display: grid;
    /* MÁGICA ACONTECE AQUI */
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 15px;
}
.galeria-item img {
    width: 100%;
    height: 200px;
    object-fit: cover; /* Evita distorção da imagem */
    border-radius: 5px;
}

Carrossel de Imagens (HTML/CSS/JS)

Um carrossel de slides simples e funcional.

HTML

<div id="meu-carrossel" class="carrossel">
    <!-- A "janela" que esconde o resto -->
    <div class="carrossel-viewport">
        <!-- O "filme" que se move (um container flex) -->
        <div class="carrossel-filmstrip">
            <!-- Cada slide -->
            <div class="carrossel-slide">
                <img src="imagem1.jpg" alt="...">
            </div>
            <div class="carrossel-slide">
                <img src="imagem2.jpg" alt="...">
            </div>
        </div>
    </div>
    <!-- Botões de navegação -->
    <button class="carrossel-btn prev"></button>
    <button class="carrossel-btn next">>/button>
</div>

CSS

.carrossel {
    position: relative; /* Ponto de referência para os botões */
    width: 100%;
}
.carrossel-viewport {
    overflow: hidden; /* <-- A MÁGICA DE ESCONDER */
    width: 100%;
}
.carrossel-filmstrip {
    display: flex;
    transition: transform 0.4s ease-in-out; /* A MÁGICA DA ANIMAÇÃO */
}
.carrossel-slide {
    flex: 0 0 100%; /* Cada slide ocupa 100% do viewport */
    width: 100%;
}
.carrossel-slide img {
    width: 100%;
    display: block;
}
.carrossel-btn {
    position: absolute; /* Fica por cima do carrossel */
    top: 50%;
    transform: translateY(-50%);
    /* ... (estilos de botão) ... */
}
.carrossel-btn.prev { left: 10px; }
.carrossel-btn.next { right: 10px; }

JavaScript (A Lógica)

// NOTA: O script completo está no fim do <body>
// Lógica principal:
document.addEventListener('DOMContentLoaded', () => {
    
    // ... (O código do carrossel está lá) ...
    
    // 1. Pega os elementos: filmstrip, prevBtn, nextBtn, slides
    // 2. Cria uma variável 'currentIndex' (começa em 0)
    // 3. Cria uma função 'goToSlide(index)'
    //    - Dentro dela: filmstrip.style.transform = `translateX(-${index * 100}%)`
    
    // 4. No clique do 'nextBtn':
    //    - Aumenta 'currentIndex' (currentIndex++)
    //    - Se 'currentIndex' > (total de slides - 1), volta para 0
    //    - Chama goToSlide(currentIndex)
    
    // 5. No clique do 'prevBtn':
    //    - Diminui 'currentIndex' (currentIndex--)
    //    - Se 'currentIndex' < 0, vai para o último slide
    //    - Chama goToSlide(currentIndex)
});
