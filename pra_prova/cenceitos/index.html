1. Introdução: A Tríade da Web

Diferente de Python ou JavaScript (que são linguagens de programação), HTML e CSS são linguagens de marcação e estilo. Elas formam a base de todas as páginas web.

  • HTML (HyperText Markup Language): Fornece a estrutura e o significado do conteúdo (o "esqueleto").
  • CSS (Cascading Style Sheets): Cuida da apresentação e estilo visual (a "pele" e "roupas").
  • JavaScript: Adiciona comportamento e interatividade (os "músculos" e "cérebro").

2. HTML: Fundamentos (As "Variáveis" da Web)

Em programação, variáveis armazenam dados. Em HTML, as tags (elementos) "armazenam" o conteúdo.

2.1. Anatomia de um Documento HTML

Todo arquivo .html segue esta estrutura básica:

<!DOCTYPE html>
<html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <title>Título da Página</title>
        <!-- Metadados, links para CSS, etc. -->
    </head>
    <body>
        <!-- Conteúdo visível da página -->
    </body>
</html>

2.2. Tags Essenciais (Os "Tipos de Dados")

Assim como há tipos de dados (int, string, bool), há tipos de tags para diferentes conteúdos:

  • <h1> a <h6>: Cabeçalhos (Títulos e subtítulos).
  • <p>: Parágrafo (o "tipo string" principal).
  • <a>: Âncora (Link).
  • <img>: Imagem.
  • <div>: Divisor genérico (um "contêiner" sem significado).
  • <span>: Contêiner genérico para texto (inline).

2.3. Atributos

Atributos fornecem informações extras sobre uma tag. Eles ficam dentro da tag de abertura.

<!-- 'href' é o atributo que define o destino do link -->
<a href="https://www.google.com">Ir para o Google</a>

<!-- 'src' (source) define o caminho da imagem -->
<img src="logo.png" alt="Logo da empresa">

<!-- 'class' e 'id' são atributos para o CSS -->
<p class="destaque" id="paragrafo-inicial">...</p>

3. HTML: Estruturas de Conteúdo

Equivalente às estruturas de dados (listas, matrizes) em programação, o HTML possui tags para agrupar e estruturar informações.

3.1. Listas (O equivalente a Listas/Arrays)

Listas Não Ordenadas (<ul>)

Para itens sem ordem específica (bullet points).

<ul>
    <li>Maçã</li>
    <li>Banana</li>
    <li>Laranja</li>
</ul>

Listas Ordenadas (<ol>)

Para itens com ordem específica (1, 2, 3...).

<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
</ol>

3.2. Tabelas (O equivalente a Matrizes)

Para exibir dados em formato de linhas e colunas (como uma matriz ou planilha).

<table>
    <thead> <!-- Cabeçalho da Tabela -->
        <tr> <!-- Linha da Tabela (Table Row) -->
            <th>Nome</th> <!-- Célula de Cabeçalho (Table Header) -->
            <th>Idade</th>
        </tr>
    </thead>
    <tbody> <!-- Corpo da Tabela -->
        <tr>
            <td>Ana</td> <!-- Célula de Dado (Table Data) -->
            <td>25</td>
        </tr>
        <tr>
            <td>Bruno</td>
            <td>30</td>
        </tr>
    </tbody>
</table>

3.3. Layout Semântico (Estruturando o Algoritmo)

Em vez de usar apenas <div> para tudo, o HTML5 fornece tags que descrevem o *propósito* de cada bloco. Isso ajuda na acessibilidade e SEO.

<body>
    <header> <!-- Cabeçalho do site (logo, menu) -->
        <nav>...</nav>
    </header>

    <main> <!-- Conteúdo principal da página -->
        <article>
            <h1>Título do Artigo</h1>
            <p>...</p>
        </article>

        <aside> <!-- Barra lateral (conteúdo relacionado) -->
            ...
        </aside>
    </main>

    <footer> <!-- Rodapé do site -->
        <p>© 2025 Meu Site</p>
    </footer>
</body>

3.4. Formulários (Entrada de Dados)

O equivalente ao input() do Python. Formulários são como o HTML coleta dados do usuário.

<form action="/submit" method="post">
    <div>
        <label for="nome">Nome:</label>
        <input type="text" id="nome" name="usuario_nome">
    </div>

    <div>
        <label for="senha">Senha:</label>
        <input type="password" id="senha" name="usuario_senha">
    </div>

    <button type="submit">Enviar</button>
</form>

4. CSS: Fundamentos (Os "Operadores")

O CSS aplica "operações" (estilos) aos elementos HTML.

4.1. Como Adicionar CSS

  1. Externo (Recomendado): Em um arquivo .css separado, linkado no <head>. 
    <link rel="stylesheet" href="estilos.css">
  2. Interno: Dentro da tag <style> no <head> do HTML. 
    <style> body { background-color: blue; } </style>
  3. Inline: Diretamente em uma tag HTML (Não recomendado). 
    <p style="color: red;">Texto vermelho.</p>

4.2. Anatomia de uma Regra CSS

Uma regra CSS é composta por um Seletor e um Bloco de Declaração.

/* Seletor     Propriedade   Valor
     |             |           |       */
    p {
        color: blue;
        font-size: 16px;
    }

4.3. Propriedades Essenciais (Manipulação)

  • color: Muda a cor do texto.
  • background-color: Muda a cor do fundo.
  • font-size: Muda o tamanho do texto.
  • font-family: Muda a fonte do texto (ex: Arial, sans-serif).
  • font-weight: Muda o "peso" da fonte (ex: bold para negrito).
  • text-align: Alinha o texto (ex: center, left, right).

5. CSS: Controle de Fluxo (Layout)

Em programação, você usa `if` e `loops` para controlar o fluxo. Em CSS, você usa **Seletores** para *decidir* quais elementos estilizar, e **Layout** para *controlar* onde eles aparecem.

5.1. Seletores (As "Condições" if)

Seletores "decidem" SE uma regra de estilo será aplicada.

  • Seletor de Tipo (Tag): Aplica a todas as tags daquele tipo. 
    /* 'if (tag === p)' */
p { color: gray; }
  • Seletor de Classe (.): O mais comum. Aplica a qualquer elemento com class="nome-da-classe". (Equivalente a uma "Função" reutilizável). 
    /* 'if (elemento.class === "destaque")' */
.destaque { 
    background-color: yellow;
    font-weight: bold;
}
    <p class="destaque">Este parágrafo é importante.</p>
<div class="destaque">Este box também é.</div>
  • Seletor de ID (#): Aplica a UM ÚNICO elemento com id="nome-do-id". IDs devem ser únicos na página. 
    /* 'if (elemento.id === "menu-principal")' */
#menu-principal { width: 100%; }
    <nav id="menu-principal">...</nav>
  • Pseudo-classes (:): Aplica estilo baseado em um *estado*. 
    /* 'if (mouse está em cima do link)' */
a:hover { text-decoration: underline; }

5.2. O Box Model (O "Algoritmo" do Espaçamento)

Todo elemento em HTML é uma caixa retangular. O Box Model é o algoritmo que define o tamanho dessa caixa.

  • content: O conteúdo (texto, imagem).
  • padding: Espaçamento interno (entre o conteúdo e a borda).
  • border: A borda da caixa.
  • margin: Espaçamento externo (entre esta caixa e outras).
.meu-box {
    width: 300px;  /* Largura do conteúdo */
    padding: 20px; /* 20px de espaço interno em todos os lados */
    border: 1px solid black; /* Borda preta */
    margin: 10px; /* 10px de espaço fora da caixa */
}

5.3. Layout com Flexbox

Flexbox é um modelo de layout para organizar itens em uma única dimensão (linha ou coluna). É a forma moderna de alinhar elementos, substituindo gambiarras antigas.

Para usar, você declara um contêiner (pai) como display: flex e alinha os itens (filhos) dentro dele.

<div class="container-flex">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
.container-flex {
    display: flex; /* Ativa o Flexbox */
    flex-direction: row; /* (Opcional) Itens em linha */
    justify-content: space-around; /* Alinha horizontalmente */
    align-items: center; /* Alinha verticalmente */
    border: 1px solid gray;
    height: 100px;
}

5.4. Layout com Grid

Grid é um modelo de layout para organizar itens em duas dimensões (linhas E colunas). É perfeito para layouts de página complexos (como matrizes).

.container-grid {
    display: grid; /* Ativa o Grid */
    /* Cria 3 colunas de tamanho igual */
    grid-template-columns: 1fr 1fr 1fr; 
    gap: 10px; /* Espaçamento entre os itens */
}

6. CSS: Tópicos Avançados (Programação em CSS)

6.1. Variáveis CSS (As "Variáveis" Reais)

Assim como em Python ou JS, você pode declarar variáveis (chamadas de Propriedades Customizadas) para reutilizar valores (como cores ou tamanhos) em todo o seu site.

/* 1. Declara a variável no seletor :root (global) */
:root {
    --cor-principal: #007bff;
    --tamanho-fonte: 16px;
}

/* 2. Usa a variável com a função var() */
body {
    font-size: var(--tamanho-fonte);
}
a {
    color: var(--cor-principal);
}
button {
    background-color: var(--cor-principal);
}

6.2. Media Queries (O "if" do CSS)

Media Queries são o equivalente a uma estrutura if. Elas aplicam estilos CSS *apenas se* uma condição (como o tamanho da tela) for verdadeira. Isso é a base do **Design Responsivo**.

/* Estilo padrão (para telas pequenas/mobile) */
.container {
    width: 90%;
    flex-direction: column;
}

/* 'if (largura da tela >= 768px)' */
@media (min-width: 768px) {
    /* Estilos para telas maiores (tablets/desktops) */
    .container {
        width: 750px;
        flex-direction: row;
    }
}

7. Exemplo Completo: HTML + CSS

Aqui está um exemplo simples que junta tudo: HTML semântico, classes, CSS externo (imaginário), Flexbox e Media Queries.

index.html (Arquivo HTML)

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Meu Site</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header class="site-header">
        <h1>Logo</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">Sobre</a></li>
            </ul>
        </nav>
    </header>

    <main class="content-area">
        <article>
            <h2>Artigo Principal</h2>
            <p>Este é o conteúdo principal.</p>
        </article>
        <aside class="sidebar">
            <h3>Barra Lateral</h3>
            <p>Links e anúncios.</p>
        </aside>
    </main>
</body>
</html>

style.css (Arquivo CSS)

/* 1. Variáveis Globais */
:root {
    --cor-principal: #333;
    --cor-links: #007bff;
}

body {
    font-family: Arial, sans-serif;
    margin: 0;
    color: var(--cor-principal);
}

.site-header {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 10px;
    background-color: #f1f1f1;
}

.content-area {
    display: flex;
    flex-direction: column; /* Mobile-first: empilhado */
    padding: 10px;
}

/* 2. Media Query (O "if") */
@media (min-width: 800px) {
    .content-area {
        flex-direction: row; /* Desktop: lado a lado */
    }
    .sidebar {
        width: 30%;
        margin-left: 20px;
    }
}
