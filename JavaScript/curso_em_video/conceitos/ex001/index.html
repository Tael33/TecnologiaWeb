1. O que é JavaScript?

JavaScript (frequentemente abreviado como JS) é uma linguagem de programação de alto nível, interpretada e dinâmica. Ela é uma das três principais tecnologias da World Wide Web, ao lado do HTML e do CSS.

  • HTML: Fornece a estrutura da página (títulos, parágrafos, imagens).
  • CSS: Cuida da apresentação e estilo (cores, fontes, layout).
  • JavaScript: Adiciona comportamento e interatividade (menus que abrem, formulários que validam, conteúdo que muda dinamicamente).

Originalmente criado para rodar apenas nos navegadores (client-side), o JavaScript hoje também é usado no lado do servidor (server-side) graças a ambientes como o Node.js.

2. Como Adicionar JS a uma Página

Existem três formas de incluir JavaScript em um arquivo HTML:

2.1. Script Externo (Recomendado)

Você cria um arquivo separado com a extensão .js e o vincula ao seu HTML. Isso mantém o código organizado e permite que o navegador armazene o script em cache.

<!-- Colocado geralmente antes de fechar a tag </body> -->
<script src="meu-script.js"></script>

2.2. Script Interno

Você pode colocar o código diretamente no HTML dentro da tag <script>.

<script>
    alert("Olá, mundo!");
</script>

2.3. Inline (Não recomendado para código complexo)

O código é colocado diretamente em um atributo de um elemento HTML, como onclick.

<button onclick="alert('Você clicou!')">Clique em mim</button>

3. Fundamentos da Linguagem

3.1. Variáveis (var, let, const)

Variáveis são "contêineres" para armazenar valores.

  • var: A forma antiga. Tem escopo de função e pode ser "hoisted" (içada). Evite seu uso em código moderno.
  • let: A forma moderna. Tem escopo de bloco ({}) e seu valor pode ser reatribuído.
  • const: A forma moderna. Tem escopo de bloco e seu valor não pode ser reatribuído após a declaração (é uma constante).
// Antigo (evitar)
var nome = "Maria";

// Moderno
let idade = 30;
idade = 31; // Válido

const PI = 3.14159;
// PI = 3.14; // Inválido! Gerará um erro.

3.2. Tipos de Dados

O JavaScript tem vários tipos de dados primitivos:

  • String: Texto. Ex: "Olá", 'Mundo'.
  • Number: Números, sejam inteiros ou decimais. Ex: 10, 2.5.
  • Boolean: Valor lógico, verdadeiro ou falso. Ex: true, false.
  • null: Representa a ausência intencional de um valor.
  • undefined: Uma variável que foi declarada mas ainda não teve um valor atribuído.
  • Symbol: (ES6+) Um valor único e imutável.
  • BigInt: (ES2020+) Para números inteiros maiores do que o Number pode suportar.

Além dos primitivos, existe o tipo Object, que é usado para coleções complexas de dados.

3.3. Operadores

São símbolos que realizam operações.

  • Aritméticos: + (adição), - (subtração), * (multiplicação), / (divisão), % (módulo/resto).
  • Atribuição: = (atribui valor), += (adiciona e atribui), -= (subtrai e atribui).
  • Comparação:
    • ==: Igual (compara valor, mas faz coerção de tipo). Ex: 5 == "5" é true.
    • ===: Estritamente igual (compara valor E tipo). Ex: 5 === "5" é false. (Prefira este!)
    • !=: Diferente.
    • !==: Estritamente diferente. (Prefira este!)
    • > (maior que), < (menor que), >= (maior ou igual), <= (menor ou igual).
  • Lógicos: && (E lógico), || (OU lógico), ! (NÃO lógico).

4. Estruturas de Controle

4.1. Condicionais (if, else, switch)

Permitem que o seu código tome decisões.

let temperatura = 25;

if (temperatura > 30) {
    console.log("Está muito quente!");
} else if (temperatura > 20) {
    console.log("Está agradável.");
} else {
    console.log("Está frio.");
}

// Switch (para múltiplas checagens de um mesmo valor)
let dia = new Date().getDay(); // Retorna 0 (Domingo) a 6 (Sábado)
let nomeDia;

switch (dia) {
    case 0:
        nomeDia = "Domingo";
        break;
    case 1:
        nomeDia = "Segunda";
        break;
    // ... outros dias
    default:
        nomeDia = "Sábado";
}

4.2. Laços de Repetição (for, while)

Permitem executar um bloco de código várias vezes.

Loop `for`

Usado quando você sabe o número de iterações.

// Conta de 0 a 4
for (let i = 0; i < 5; i++) {
    console.log("O número é " + i);
}

Loop `while`

Usado quando a condição de parada é baseada em algo que muda dentro do loop.

let contador = 0;
while (contador < 5) {
    console.log("Contador: " + contador);
    contador++;
}

5. Funções

Funções são blocos de código reutilizáveis. Elas podem receber dados (parâmetros) e retornar um valor.

Declaração de Função (Padrão)

function saudar(nome) {
    return "Olá, " + nome + "!";
}

let saudacao = saudar("Ana"); // saudacao agora é "Olá, Ana!"
console.log(saudacao);

Expressão de Função (Anônima)

const somar = function(a, b) {
    return a + b;
};

let resultado = somar(5, 3); // resultado é 8
console.log(resultado);

Veja sobre Arrow Functions na seção JavaScript Moderno.

6. Objetos e Arrays

Objetos

Coleções de pares chave-valor. Usado para agrupar dados relacionados.

const pessoa = {
    nome: "Carlos",
    sobrenome: "Silva",
    idade: 42,
    ativo: true,
    // Um método (função dentro de um objeto)
    nomeCompleto: function() {
        return this.nome + " " + this.sobrenome;
    }
};

// Acessando propriedades
console.log(pessoa.nome); // "Carlos"
console.log(pessoa["idade"]); // 42
console.log(pessoa.nomeCompleto()); // "Carlos Silva"

Arrays (Matrizes)

Listas ordenadas de valores. Os índices começam em 0.

const frutas = ["Maçã", "Banana", "Laranja"];

// Acessando valores
console.log(frutas[0]); // "Maçã"
console.log(frutas[2]); // "Laranja"

// Adicionando um item no final
frutas.push("Uva");
console.log(frutas); // ["Maçã", "Banana", "Laranja", "Uva"]

// Iterando sobre um array
frutas.forEach(function(fruta) {
    console.log("Eu gosto de " + fruta);
});

7. O DOM (Document Object Model)

O DOM é a parte mais importante do JavaScript no navegador. É uma representação em forma de árvore (objeto) do seu documento HTML. O JavaScript usa o DOM para "enxergar" e "modificar" o HTML e o CSS.

7.1. Selecionando Elementos

Para manipular um elemento, você primeiro precisa selecioná-lo.

// Seleciona um elemento pelo seu ID
const titulo = document.getElementById("meu-titulo");

// Seleciona o *primeiro* elemento que bate com um seletor CSS
const primeiroBotao = document.querySelector(".btn-comprar");

// Seleciona *todos* os elementos que batem com um seletor CSS
const todosOsLinks = document.querySelectorAll("a");

// Seleciona pela classe (antigo, retorna uma HTMLCollection)
const paragrafos = document.getElementsByClassName("paragrafo");
Nota: querySelector (para um elemento) e querySelectorAll (para múltiplos) são os métodos modernos e preferidos.

7.2. Manipulando Elementos

Uma vez selecionado, você pode mudar seu conteúdo, estilo ou atributos.

// Suponha que temos: 
Olá

const titulo = document.getElementById("titulo-principal");

// Mudar o texto
titulo.textContent = "Bem-vindo ao JavaScript!";

// Mudar o HTML interno
// Cuidado: usar .innerHTML pode ser um risco de segurança (XSS) se você inserir dados do usuário.
titulo.innerHTML = "Bem-vindo ao JavaScript!";

// Mudar o estilo (CSS)
titulo.style.color = "blue";
titulo.style.fontSize = "24px";

// Mudar atributos (ex: um link)
const link = document.querySelector("a");
link.setAttribute("href", "https://www.google.com");

// Criar e adicionar elementos
const novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Este parágrafo foi criado com JS.";
document.body.appendChild(novoParagrafo); // Adiciona no final do

8. Eventos

Eventos são ações que acontecem na página (um clique, um envio de formulário, uma tecla pressionada). O JavaScript pode "ouvir" (listen) esses eventos e executar uma função (um "callback") quando eles ocorrem.

A forma moderna de fazer isso é com addEventListener.

<!-- No seu HTML -->
<button id="meu-botao">Clique Aqui!</button>

<!-- No seu JavaScript -->
const botao = document.getElementById("meu-botao");

botao.addEventListener("click", function() {
    // Este código só roda quando o botão é clicado
    alert("O botão foi clicado!");
    document.body.style.backgroundColor = "yellow";
});

// Outros eventos comuns:
// "mouseover": quando o mouse passa por cima
// "keydown": quando uma tecla é pressionada
// "submit": quando um formulário é enviado

9. JavaScript Moderno (ES6+)

Em 2015, o JavaScript recebeu uma grande atualização chamada ECMAScript 2015 (ou ES6), que modernizou a linguagem.

9.1. Arrow Functions (Funções de Seta)

Uma sintaxe mais curta e limpa para escrever funções.

// Função normal
const somar = function(a, b) {
    return a + b;
};

// Arrow Function (equivalente)
const somarArrow = (a, b) => {
    return a + b;
};

// Se tiver apenas uma linha de retorno, pode ser mais curto
const somarCurto = (a, b) => a + b;

// Muito usado em callbacks
const numeros = [1, 2, 3];
const dobrados = numeros.map(n => n * 2); // [2, 4, 6]

9.2. Promises e Async/Await

JavaScript é "assíncrono". Isso significa que ele pode iniciar uma tarefa (como buscar dados de um servidor) e continuar executando outro código sem esperar a tarefa terminar.

Promises são objetos que representam a eventual conclusão (ou falha) de uma operação assíncrona.

async/await é uma sintaxe moderna (ES2017) que torna o código assíncrono mais fácil de ler, fazendo-o parecer síncrono. 

// "async" na frente da função diz que ela pode usar "await"
async function buscarDadosDaAPI() {
    try {
        console.log("Buscando dados...");
        // "await" pausa a execução da função até que a Promise (fetch) seja resolvida
        const resposta = await fetch("https://api.exemplo.com/dados");
        
        // Esta linha só executa após o 'await' acima
        const dados = await resposta.json(); 

        console.log("Dados recebidos:", dados);
    } catch (erro) {
        // 'catch' pega qualquer erro que ocorra no 'try'
        console.error("Falha ao buscar dados:", erro);
    }
}

buscarDadosDaAPI();
console.log("Este log aparece ANTES dos dados serem recebidos!");

10. Próximos Passos

Isto foi apenas uma visão geral. O universo JavaScript é vasto. A partir daqui, você pode explorar:

  • Frameworks e Bibliotecas: Ferramentas que facilitam a criação de aplicações complexas.
    • React: Biblioteca popular (criada pelo Facebook) para construir interfaces de usuário.
    • Angular: Framework completo (criado pelo Google) para aplicações robustas.
    • Vue.js: Um framework progressivo, conhecido por sua curva de aprendizado mais suave.
  • Node.js: Para usar JavaScript no back-end (criar servidores, APIs, scripts de automação).
  • TypeScript: Um "superset" do JavaScript que adiciona tipos estáticos, ajudando a prevenir bugs em projetos grandes.
