1. O que é JavaScript?

JavaScript (JS) é uma linguagem de programação de alto nível, interpretada e dinâmica. É uma das três principais tecnologias da World Wide Web:

  • HTML: Fornece a estrutura (o "esqueleto").
  • CSS: Cuida da apresentação e estilo (a "pele" e "roupas").
  • JavaScript: Adiciona comportamento e interatividade (os "músculos" e "cérebro").

2. Como Adicionar JS a uma Página

2.1. Script Externo (Recomendado)

Crie um arquivo .js e o vincule ao HTML. Isso mantém o código organizado e permite cache.

<!-- Colocado geralmente antes de fechar a tag </body> -->
<script src="meu-script.js"></script>

2.2. Script Interno

Código diretamente no HTML dentro da tag <script>.

<script>
    console.log("Olá, mundo!");
</script>

3. Fundamentos de Algoritmos (Revisão)

Esta seção cobre os conceitos básicos de algoritmos, adaptados do PDF para JavaScript.

3.1. Saída e Entrada de Dados

Saída de Dados (Impressão)

Em Python, usa-se print()[cite: 1209]. Em JavaScript, a forma mais comum de "imprimir" valores para depuração é usando o console do navegador.

console.log("Hello, World!");
let nome = "Ana";
console.log("Olá,", nome, "Silva"); // Imprime: Olá, Ana Silva

Para exibir dados na própria página, você usa a Manipulação do DOM.

Entrada de Dados (Leitura)

Em Python, usa-se input()[cite: 1268]. Em JavaScript, a forma mais simples (e antiga) de pedir dados ao usuário é com prompt().

Nota: prompt() pausa todo o script e não é recomendado para aplicações reais. A forma moderna é usar campos de formulário HTML (<input>) e Eventos. 
// 1. Entrada de String
let nomeUsuario = prompt("Digite o nome: ");
console.log("Nome: ", nomeUsuario);

// 2. Entrada de Números
// prompt() SEMPRE retorna uma string[cite: 1268]. É preciso converter.
let idadeString = prompt("Digite a idade: ");
let idade = Number(idadeString); // Converte string para número

let base = Number(prompt("Digite o valor da base: "));
let altura = Number(prompt("Digite o valor da altura: "));
let area = (base * altura) / 2;
console.log("Área:", area);

3.2. Variáveis

Variáveis são "contêineres" para armazenar valores[cite: 1213].

  • let: Para valores que podem mudar. Tem escopo de bloco ({}).
  • const: Para valores que não podem mudar (constantes). Tem escopo de bloco.
  • var: A forma antiga. Tem escopo de função. Evite seu uso em código moderno.
let idade = 30;
idade = 31; // Válido

const PI = 3.14159;
// PI = 3.14; // Inválido! Gerará um erro.

3.3. Tipos de Dados

O JavaScript tem tipos de dados primitivos e o tipo objeto.

Primitivos

  • String: Texto. Ex: "Olá", 'Mundo'. (Equivalente ao str do Python [cite: 1216])
  • Number: Números, inteiros ou decimais. Ex: 10, 2.5. (Equivalente ao int [cite: 1214] e float [cite: 1215] do Python)
  • Boolean: Valor lógico. Ex: true, false. (Equivalente ao bool do Python [cite: 1217])
  • null: Representa a ausência intencional de um valor.
  • undefined: Uma variável que foi declarada mas ainda não teve um valor atribuído.

Não-Primitivo

  • Object: Coleções complexas de dados. Arrays, Funções e Objetos literais são todos do tipo Objeto.

3.4. Operadores

São símbolos que realizam operações. A precedência é similar à do Python (parênteses, exponencial, multiplicação/divisão, adição/subtração)[cite: 1263, 1264, 1265, 1266].

Operadores Aritméticos Puros

Estes operadores realizam um cálculo e retornam um novo valor.

  • + (Adição)
    • Soma dois números ou concatena duas strings.
    • Exemplo (Número): 5 + 3 resulta em 8.
  • - (Subtração)
    • Subtrai o segundo número do primeiro.
    • Exemplo: 10 - 4 resulta em 6.
  • * (Multiplicação)
    • Multiplica dois números.
    • Exemplo: 7 * 3 resulta em 21.
  • / (Divisão)
    • Divide o primeiro número pelo segundo.
    • Exemplo: 10 / 2 resulta em 5.
  • % (Módulo / Resto)
    • Retorna o resto da divisão.
    • Exemplo: 10 % 3 resulta em 1.
  • ** (Exponenciação - ES6)
    • Eleva o primeiro número à potência do segundo.
    • Exemplo: 2 ** 3 resulta em 8.

Operadores de Atribuição (Auto-atribuição)

Modificam o valor da própria variável.

Incremento e Decremento
  • ++ (Incremento)
    • Adiciona 1 ao valor (x = x + 1).
    • Exemplo: let x = 5; x++; // x agora é 6.
  • -- (Decremento)
    • Subtrai 1 do valor (x = x - 1).
    • Exemplo: let x = 5; x--; // x agora é 4.
Atribuição Composta
  • += (x = x + y)
  • -= (x = x - y)
  • *= (x = x * y)
  • /= (x = x / y)
  • %= (x = x % y)
  • **= (x = x ** y)

Operadores de Comparação (Relacionais)

Comparam dois valores e retornam true ou false[cite: 1327].

  • === (Igualdade Estrita)
    • Equivalente ao == do Python[cite: 1328]. Compara valor E tipo. (Recomendado).
    • Exemplo: 5 === "5" resulta em false.
  • !== (Diferença Estrita)
    • Equivalente ao != do Python[cite: 1329]. Compara valor E tipo. (Recomendado).
    • Exemplo: 5 !== "5" resulta em true.
  • > (Maior que) [cite: 1330]
  • < (Menor que) [cite: 1331]
  • >= (Maior ou igual a) [cite: 1332]
  • <= (Menor ou igual a) [cite: 1333]
  • == (Igualdade Solta)
    • Compara só o valor, fazendo coerção. Evite este operador.
    • Exemplo: 5 == "5" resulta em true.

Operadores Lógicos

Usados para combinar expressões.

  • && (E Lógico - AND)
    • Equivalente ao and do Python[cite: 1335]. Retorna true se AMBAS forem verdadeiras.
    • Exemplo: (idade > 18) && (temCNH === true)
  • || (OU Lógico - OR)
    • Equivalente ao or do Python[cite: 1336]. Retorna true se UMA for verdadeira.
    • Exemplo: (deficiente === "sim") || (idade >= 65) [cite: 1387]
  • ! (NÃO Lógico - NOT)
    • Equivalente ao not do Python[cite: 1337]. Inverte o valor.
    • Exemplo: !isValid

3.5. Escopo (Scope) e Hoisting

Escopo

Escopo define onde as variáveis e funções são acessíveis no seu código.

  • Escopo Global: Declarado fora de qualquer função. Acessível de qualquer lugar.
  • Escopo de Função: Variáveis declaradas com var são acessíveis apenas dentro da função.
  • Escopo de Bloco: Variáveis com let e const são acessíveis apenas dentro do bloco ({}) onde foram criadas.

Hoisting (Içamento)

Hoisting é o comportamento do JavaScript de mover declarações de variáveis (com var) e funções para o topo do seu escopo antes da execução. let e const também sofrem hoisting, mas não são inicializadas, causando um erro se acessadas antes.

3.6. Coerção de Tipo (Type Coercion)

Coerção é a conversão automática de um tipo de dado para outro. 

console.log(10 + "5");   // "105" (Número vira String)
console.log(10 - "5");   // 5 (String vira Número)
console.log("5" == 5);   // true (Coerção de tipo)
console.log("5" === 5);  // false (Sem coerção)

3.7. Template Literals (ES6)

Uma forma moderna de criar strings com variáveis (interpolação). Elas usam crases (``).

const nome = "Ana";
const idade = 28;

// Forma moderna (Template Literal)
const saudacaoModerna = `Olá, meu nome é ${nome} e eu tenho ${idade} anos.`;

4. Estruturas de Controle

4.1. Condicionais (if, else if, else)

Permite que o programa execute blocos de código com base em condições[cite: 1322]. Em JavaScript, o elif do Python [cite: 1325] é escrito como else if.

let num = -5;

if (num > 0) {
    console.log("O número é positivo.");
} else if (num < 0) {
    console.log("O número é negativo.");
} else {
    console.log("O número é zero.");
}
// Este código é o equivalente em JS do Código 1.12 do PDF[cite: 1339, 1340, 1341, 1342, 1343, 1344].

// Exemplo do PDF [cite: 1372] traduzido para JS
let idade = 18;

if (idade <= 10) {
    console.log("Categoria A");
} else if (idade <= 20) {
    console.log("Categoria B");
} else {
    console.log("Categoria C");
}
// Saída: Categoria B

4.2. Laços de Repetição (for, while)

Loop `while`

No while, enquanto a condição for verdadeira, os comandos serão executados[cite: 1422].

// Equivalente ao Código 1.18 do PDF [cite: 1427, 1428, 1429, 1430, 1431, 1432, 1433, 1434]
let cont = 0;
while (cont <= 5) {
    console.log(cont);
    cont = cont + 1; // Ou cont++;
}
// Saída: 0, 1, 2, 3, 4, 5

// Loop com condição de parada (similar ao Código 1.21 do PDF [cite: 1490])
let idade = Number(prompt('Digite a idade (ou 0 para sair):'));
let somaIdades = 0;
let qtPessoas = 0;

while (idade > 0) {
    somaIdades = somaIdades + idade;
    qtPessoas = qtPessoas + 1;
    idade = Number(prompt('Digite a idade (ou 0 para sair):'));
}

if (qtPessoas > 0) {
    let media = somaIdades / qtPessoas;
    console.log('Média das idades:', media);
} else {
    console.log('Nenhuma idade foi digitada.');
}

Loop `for`

O for clássico é usado para repetir um número específico de vezes. O for...of é usado para percorrer listas/arrays.

// Loop `for` clássico (contagem)
for (let i = 0; i < 5; i++) {
    console.log("O número é " + i);
}

// Loop `for...of` (para iterar sobre valores de arrays)
// Equivalente a "Percorrendo listas com FOR" do PDF 
const frutas = ["Maçã", "Banana", "Laranja"];
for (const fruta of frutas) {
    console.log(fruta);
}

5. Estruturas de Dados

5.1. Arrays (Listas)

Em JavaScript, Arrays são o equivalente às Listas do Python. São listas ordenadas de valores, e os índices começam em 0.

// Em Python: lista = [10, 20, 30]
const meuArray = [10, 20, 30];

// Acessando valores
console.log(meuArray[0]); // 10

// Alterando um valor
meuArray[1] = 25;
console.log(meuArray); // [10, 25, 30]

// Adicionando um item no final (similar ao .append())
meuArray.push(40);
console.log(meuArray); // [10, 25, 30, 40]

// Removendo do final
meuArray.pop();
console.log(meuArray); // [10, 25, 30]

// Tamanho da lista
console.log(meuArray.length); // 3

5.2. Arrays Aninhados (Matrizes)

Assim como em Python, matrizes são implementadas como "listas dentro de listas", ou, em JS, "arrays dentro de arrays".

// Em Python: matriz = [[1, 2], [3, 4]]
const matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

// Acessando um elemento (linha 1, coluna 2)
console.log(matriz[1][2]); // 6

// Iterando sobre a matriz (similar ao PDF )
for (let i = 0; i < matriz.length; i++) {
    // i é o índice da linha
    for (let j = 0; j < matriz[i].length; j++) {
        // j é o índice da coluna
        console.log(`Linha ${i}, Coluna ${j}: ${matriz[i][j]}`);
    }
}

5.3. Métodos de Array (Métodos em Listas)

JavaScript tem métodos poderosos para manipular arrays, assim como Python tem métodos para listas.

.map()

Transforma cada item de um array e retorna um novo array.

const numeros = [1, 2, 3, 4];
const dobrados = numeros.map(n => n * 2);
// dobrados é [2, 4, 6, 8]

.filter()

Cria um novo array contendo apenas os itens que passam em um teste.

const numeros = [1, 2, 3, 4, 5];
const pares = numeros.filter(n => n % 2 === 0);
// pares é [2, 4]

.reduce()

Reduz o array a um único valor (ex: uma soma).

const numeros = [1, 2, 3, 4, 5];
const soma = numeros.reduce((total, n) => total + n, 0);
// soma é 15

5.4. Objetos (Dicionários)

Em JavaScript, Objetos são o equivalente principal aos Dicionários do Python. Eles armazenam coleções de pares chave-valor.

// Em Python: pessoa = { "nome": "Carlos", "idade": 42 } 
const pessoa = {
    nome: "Carlos",
    idade: 42,
    ativo: true,
    endereco: { // Objetos podem ser aninhados
        cidade: "São Paulo",
        estado: "SP"
    }
};

// Acessando valores (duas formas)
console.log(pessoa.nome); // "Carlos"
console.log(pessoa["idade"]); // 42

// Alterando valores
pessoa.idade = 43;
pessoa["ativo"] = false;

// Adicionando novas chaves
pessoa.profissao = "Engenheiro";

5.5. Manipulação de Strings

Assim como o Python, JavaScript oferece muitos métodos para manipular strings.

  • .length: Retorna o tamanho da string.
  • .toUpperCase(): Converte para maiúsculas (Equivalente ao .upper() [cite: 1005]).
  • .toLowerCase(): Converte para minúsculas (Equivalente ao .lower() [cite: 1006]).
  • .trim(): Remove espaços em branco do início e do fim (Equivalente ao .strip() [cite: 1007]).
  • .includes(substr): Verifica se a string contém uma substring (retorna true/false).
  • .slice(inicio, fim): Extrai uma parte da string (Fatiamento ).
  • .split(separador): Divide a string em um array.
let frase = "  Olá Mundo!  ";
console.log(frase.length); // 14
console.log(frase.toUpperCase()); // "  OLÁ MUNDO!  "
console.log(frase.trim()); // "Olá Mundo!"
console.log(frase.trim().slice(0, 3)); // "Olá"

let dados = "ana;silva;30";
let partes = dados.split(";");
console.log(partes); // ["ana", "silva", "30"]

5.6. Desestruturação (Destructuring) e Spread

Sintaxes modernas (ES6) para facilitar a extração de dados de arrays e objetos, similar ao "desempacotamento" em Python.

Desestruturação de Objeto

const usuario = { id: 1, nome: "Helena", email: "helena@email.com" };
const { nome, email } = usuario;
console.log(nome); // "Helena"

Desestruturação de Array

const [primeiro, segundo] = ["Ouro", "Prata", "Bronze"];
console.log(primeiro); // "Ouro"

Spread Operator (...)

Usado para "espalhar" os elementos de um array ou as propriedades de um objeto.

const arr1 = [1, 2, 3];
const arrCombinado = [...arr1, 4, 5, 6]; // [1, 2, 3, 4, 5, 6]

const obj1 = { a: 1, b: 2 };
const objMesclado = { ...obj1, c: 3 }; // { a: 1, b: 2, c: 3 }

6. Funções (Functions)

Funções são blocos de código reutilizáveis que executam uma tarefa específica. Elas podem receber dados (parâmetros) e retornar um valor.

6.1. Declarando Funções

Declaração de Função (Padrão)

// Em Python: def saudar(nome):
function saudar(nome) {
    return `Olá, ${nome}!`;
}
let saudacao = saudar("Ana");
console.log(saudacao);

Expressão de Função (Anônima)

const somar = function(a, b) {
    return a + b;
};
let resultado = somar(5, 3);
console.log(resultado);

6.2. Parâmetros (Default e Rest)

Parâmetros Padrão (ES6)

Você pode definir um valor padrão para um parâmetro caso ele não seja fornecido.

function registrar(nome, plano = "BÁSICO") {
    console.log(`${nome} está no plano ${plano}.`);
}
registrar("Bia"); // "Bia está no plano BÁSICO."

Parâmetros Rest (ES6)

Permite que uma função aceite um número indefinido de argumentos como um array.

function somarTudo(...numeros) {
    // 'numeros' será um array
    return numeros.reduce((total, num) => total + num, 0);
}
console.log(somarTudo(1, 2, 3, 4)); // 10

6.3. O 'this' Keyword

O this é um conceito complexo em JS, diferente do self do Python. Seu valor é determinado por como a função é chamada.

  • Em um método de objeto: this refere-se ao próprio objeto.
  • Em uma função normal: this refere-se ao objeto global (window).
  • Em Arrow Functions: this é "herdado" do escopo onde a arrow function foi criada.

7. JS no Navegador: O DOM

Esta é a maior diferença do Python: JavaScript pode interagir diretamente com o conteúdo de uma página web. O DOM (Document Object Model) é a representação em objeto do seu HTML.

7.1. Selecionando Elementos

Para manipular um elemento, você primeiro precisa selecioná-lo.

// Seleciona o *primeiro* elemento que bate com um seletor CSS
const titulo = document.querySelector("#meu-titulo");
const primeiroBotao = document.querySelector(".btn-comprar");

// Seleciona *todos* os elementos (retorna uma NodeList)
const todosOsLinks = document.querySelectorAll("a");

7.2. Manipulando Elementos

Uma vez selecionado, você pode mudar tudo.

const titulo = document.querySelector("#titulo-principal");

// Mudar o texto
titulo.textContent = "Bem-vindo ao JavaScript!";

// Mudar o HTML interno
titulo.innerHTML = "Bem-vindo ao <i>JavaScript</i>!";

// Mudar o estilo (CSS)
titulo.style.color = "blue";
titulo.style.backgroundColor = "#f0f0f0";

// Adicionar/Remover classes CSS
titulo.classList.add("destaque");
titulo.classList.remove("antiga-classe");

8. Eventos

Eventos são ações que acontecem na página (clique, tecla pressionada, envio de formulário). O JavaScript pode "ouvir" (listen) esses eventos e executar uma função (callback).

8.1. addEventListener

A forma moderna de "ouvir" eventos.

<!-- No seu HTML -->
<button id="meu-botao">Clique Aqui!</button>

<!-- No seu JavaScript -->
const botao = document.querySelector("#meu-botao");

botao.addEventListener("click", function(event) {
    // Este código só roda quando o botão é clicado
    console.log("O botão foi clicado!");
    document.body.style.backgroundColor = "yellow";
});

8.2. Propagação de Eventos e preventDefault

event.preventDefault() impede o comportamento padrão de um elemento (como um link de navegar ou um formulário de recarregar a página).

const form = document.querySelector("#meu-form");
form.addEventListener("submit", function(event) {
    // Impede que o formulário recarregue a página
    event.preventDefault(); 
    console.log("Formulário enviado via JS!");
});

9. JavaScript Moderno (ES6+)

9.1. Arrow Functions (Funções de Seta)

Uma sintaxe mais curta para escrever funções, que também herda o this do escopo pai.

// Função normal
const somar = function(a, b) { return a + b; };

// Arrow Function
const somarArrow = (a, b) => a + b;

9.2. Promises e Async/Await

JavaScript é "assíncrono". Promises e a sintaxe async/await são usadas para lidar com operações que demoram, como buscar dados de um servidor (API), sem travar a página. 

async function buscarDadosDaAPI() {
    try {
        const resposta = await fetch("https://api.github.com/users/google");
        const dados = await resposta.json(); 
        console.log(dados.name);
    } catch (erro) {
        console.error("Falha ao buscar dados:", erro);
    }
}
buscarDadosDaAPI();

9.3. Classes (ES6)

Uma sintaxe mais limpa para criar objetos e lidar com herança (Programação Orientada a Objetos), similar às classes do Python.

class Pessoa {
    constructor(nome, idade) {
        this.nome = nome;
        this.idade = idade;
    }
    apresentar() {
        console.log(`Olá, meu nome é ${this.nome}.`);
    }
}

class Estudante extends Pessoa {
    constructor(nome, idade, curso) {
        super(nome, idade); // Chama o construtor da classe 'Pai'
        this.curso = curso;
    }
}

const e1 = new Estudante("João", 22, "Ciência da Computação");
e1.apresentar(); // "Olá, meu nome é João."

9.4. Módulos (import/export)

Permitem dividir seu código em múltiplos arquivos para melhor organização, assim como import em Python.

Nota: Para import/export funcionar no navegador, seu script HTML deve ter type="module".

utils.js (Exportando)

export function somar(a, b) {
    return a + b;
}
export default function subtrair(a, b) {
    return a - b;
}

main.js (Importando)

import { somar } from './utils.js';
import minhaSubtracao from './utils.js'; // Import default

console.log(somar(10, 5)); // 15
console.log(minhaSubtracao(10, 5)); // 5

10. Tópicos Adicionais

10.1. Manipulação de Erros (try...catch)

Permite que você "tente" (try) executar um bloco de código que pode falhar. Se falhar, o bloco catch é executado, impedindo que o script "quebre". Similar ao try...except do Python. 

try {
    let dados = JSON.parse("{ 'json_invalido': true }"); // Vai falhar
    console.log(dados);
} catch (erro) {
    console.error("Ocorreu um erro:", erro.message);
} finally {
    console.log("Execução finalizada.");
}

10.2. JSON (JavaScript Object Notation)

JSON é um formato de texto para troca de dados, muito usado em APIs. É nativo do JavaScript. 

// 1. JSON.parse() (String → Objeto JS)
const jsonString = '{ "nome": "Bruno", "id": 123 }';
const objetoJS = JSON.parse(jsonString);
console.log(objetoJS.nome); // "Bruno"

// 2. JSON.stringify() (Objeto JS → String)
const objetoJS = { nome: "Ana", ativa: true };
const jsonString = JSON.stringify(objetoJS);
console.log(jsonString); // "{\"nome\":\"Ana\",\"ativa\":true}"

10.3. Web Storage (localStorage)

Permite salvar dados simples (strings) no navegador do usuário de forma persistente. Diferente de manipulação de arquivos (como em Python ), isso é feito no lado do cliente. 

// 1. Salvar dados
localStorage.setItem("tema", "dark");

// 2. Ler dados
const temaSalvo = localStorage.getItem("tema");
console.log(temaSalvo); // "dark"

// 3. Remover um item
localStorage.removeItem("tema");

// Para salvar objetos, use JSON
const config = { volume: 80 };
localStorage.setItem("configuracoes", JSON.stringify(config));
