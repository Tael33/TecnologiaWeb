Guia Completo: A Árvore DOM

1. O que é o DOM?

O DOM (Document Object Model) é uma representação em forma de árvore do seu arquivo HTML. O navegador cria essa "árvore de objetos" para que o JavaScript possa entender e interagir com o conteúdo da página.

Cada tag HTML (<body>, <div>, <p>), cada atributo (class, id) e cada pedaço de texto é um nó (node) nessa árvore.

O JavaScript não lê o arquivo HTML. Ele lê o DOM.

Visualização da Árvore

Para este HTML:

<html>
  <head>
    <title>Meu Título</title>
  </head>
  <body>
    <h1>Olá, Mundo!</h1>
    <p id="intro">Meu parágrafo.</p>
  </body>
</html>

O navegador cria uma árvore parecida com esta:


(window)
  └── document
      └── <html>
          ├── <head>
          |   └── <title>
          |       └── (nó de texto: "Meu Título")
          |
          └── <body>
              ├── <h1>
              |   └── (nó de texto: "Olá, Mundo!")
              |
              └── <p> (com id "intro")
                  └── (nó de texto: "Meu parágrafo.")

O objetivo do JavaScript é encontrar um nó (ex: o <p> com id "intro") e fazer algo com ele (mudar seu texto, cor, etc.).

2. Seleção de Elementos (Encontrando Nós)

O primeiro passo é sempre selecionar o elemento que você quer manipular. O document é o ponto de entrada para todas as seleções.

2.1 document.querySelector(seletorCSS) (Recomendado)

O método mais moderno e versátil. Ele usa seletores CSS (os mesmos que você usa no seu arquivo .css) e retorna o primeiro elemento que combinar.

// Seleciona o primeiro elemento com o id 'intro'
const p_intro = document.querySelector('#intro');

// Seleciona o primeiro elemento com a classe 'destaque'
const item = document.querySelector('.destaque');

// Seleciona a primeira tag <h1>
const titulo = document.querySelector('h1');

// Seletores complexos também funcionam
const primeiroLink = document.querySelector('nav ul li a');

2.2 document.querySelectorAll(seletorCSS) (Recomendado)

Igual ao querySelector, mas retorna TODOS os elementos que combinam, em forma de uma NodeList (que parece um array).

// Seleciona TODOS os elementos com a classe 'item-lista'
const todosItens = document.querySelectorAll('.item-lista');

// Você pode então iterar sobre eles com forEach
todosItens.forEach((item) => {
  item.style.color = 'blue';
});

2.3 document.getElementById(id)

O método clássico para pegar um elemento pelo seu id. É muito rápido.

Nota: Você passa apenas o nome do id, sem o #.
// Seleciona o elemento <div id="menu-principal">...</div>
const menu = document.getElementById('menu-principal');

2.4 document.getElementsByClassName(classe)

Retorna TODOS os elementos com uma classe, em forma de uma HTMLCollection (parecido com array, mas um pouco mais antigo).

Nota: Você passa apenas o nome da classe, sem o ..
// Seleciona todos os <p class="aviso">...</p>
const avisos = document.getElementsByClassName('aviso');

// HTMLCollection não tem forEach. Use um loop 'for' clássico.
for (let i = 0; i < avisos.length; i++) {
  avisos[i].style.fontWeight = 'bold';
}

2.5 document.getElementsByTagName(tag)

Retorna TODOS os elementos de uma certa tag (ex: todos os <li>).

// Seleciona todas as tags <li> da página
const todosOsLinks = document.getElementsByTagName('li');

3. Manipulação de Elementos (Alterando Nós)

Depois de selecionar um elemento e guardá-lo em uma variável, você pode fazer qualquer coisa com ele.

Área de Teste Interativa

Este é o elemento que vamos manipular: Eu sou o alvo!

3.1 Alterar Conteúdo (textContent e innerHTML)

.textContent: Muda apenas o texto. É mais seguro e rápido.

const titulo = document.querySelector('h1');
titulo.textContent = 'Novo Título da Página';

.innerHTML: Muda o HTML interno. Permite inserir novas tags, mas é um risco de segurança (XSS) se você inserir dados de um usuário.

const aviso = document.getElementById('aviso');
aviso.innerHTML = '<strong>Atenção!</strong> Você foi avisado.';

3.2 Alterar Estilos (.style)

Você pode alterar qualquer propriedade CSS diretamente pelo atributo style. Propriedades com hífen (background-color) viram camelCase (backgroundColor).

const alvo = document.querySelector('#item-alvo');
alvo.style.color = 'red';
alvo.style.backgroundColor = 'yellow';
alvo.style.fontSize = '24px';

3.3 Alterar Atributos (setAttribute, getAttribute)

Usado para ler ou definir atributos HTML, como href de um link, src de uma imagem ou disabled de um botão.

const link = document.querySelector('a');
link.setAttribute('href', 'https://www.google.com');
link.setAttribute('target', '_blank');

const imagem = document.querySelector('img');
const urlAntiga = imagem.getAttribute('src');

3.4 Alterar Classes (.classList) (Recomendado)

Esta é a forma correta de alterar a aparência de um elemento. Em vez de mudar o .style direto no JS, você deve criar classes no seu CSS e apenas adicionar ou remover essas classes com JS.

  • .classList.add('nome-da-classe'): Adiciona a classe.
  • .classList.remove('nome-da-classe'): Remove a classe.
  • .classList.toggle('nome-da-classe'): Adiciona se não tiver, remove se tiver.
// CSS:
.destaque {
  background-color: yellow;
  font-weight: bold;
}

// JS:
const alvo = document.querySelector('#item-alvo');
alvo.classList.add('destaque'); // Adiciona a classe .destaque
alvo.classList.remove('classe-antiga'); // Remove a classe .classe-antiga
alvo.classList.toggle('ativo'); // Liga/Desliga a classe .ativo

4.1 createElement e appendChild

Você pode criar novos elementos do zero e adicioná-los à árvore.

// 1. Seleciona o PAI onde o novo elemento vai entrar
const lista = document.querySelector('#lista-alvo');

// 2. Cria o novo elemento (ele ainda não está na página)
const novoItem = document.createElement('li');

// 3. Configura o novo elemento
novoItem.textContent = 'Novo Item Adicionado via JS';
novoItem.classList.add('box-verde');

// 4. Adiciona o novo elemento ao PAI (agora ele aparece na página)
lista.appendChild(novoItem);

4.2 .remove() e .removeChild()

A forma moderna de remover um elemento é simplesmente chamar .remove() no próprio elemento.

// 1. Encontra o elemento que você quer remover
const itemParaRemover = document.querySelector('#lista-alvo li'); // Pega o primeiro 
  • 

// 2. Remove o elemento
if (itemParaRemover) {
  itemParaRemover.remove();
}

    • A forma clássica (ainda útil) é dizer ao PAI para remover um FILHO:

    // 1. Pega o PAI
const pai = document.querySelector('#lista-alvo');

// 2. Pega o FILHO para remover
const filho = document.querySelector('#lista-alvo li');

// 3. O PAI remove o FILHO
if (pai && filho) {
  pai.removeChild(filho);
}

    5. Navegação na Árvore

    Depois de selecionar um nó, você pode "andar" pela árvore a partir dele.

    // <div id="container">
//   <p>Irmão anterior</p>
//   <h2 id="meu-h2">Meu Título</h2> <-- Estamos aqui
//   <p>Irmão seguinte</p>
// </div>

const h2 = document.querySelector('#meu-h2');

// 1. Acessar o PAI
const container = h2.parentElement; // Pega o <div id="container">
container.style.border = '1px solid red';

// 2. Acessar os FILHOS (de um pai)
const todosFilhosDoContainer = container.children; // Retorna uma HTMLCollection [<p>, <h2>, <p>]

// 3. Acessar o IRMÃO SEGUINTE
const irmaoSeguinte = h2.nextElementSibling; // Pega o <p> "Irmão seguinte"

// 4. Acessar o IRMÃO ANTERIOR
const irmaoAnterior = h2.previousElementSibling; // Pega o <p> "Irmão anterior"

    6. Eventos (Interatividade)

    Eventos são como o DOM reage a ações do usuário (cliques, teclado, mouse, etc.).

    6.1 .addEventListener() (Recomendado)

    Esta é a forma moderna de "ouvir" um evento. Você diz a um elemento para "escutar" por uma ação (ex: 'click') e executar uma função (callback) quando ela acontecer.

    ...

    // 1. Seleciona o elemento
const botao = document.querySelector('#btn-evento');
const resultado = document.querySelector('#resultado-evento');

// 2. Adiciona o "ouvidor de evento"
botao.addEventListener('click', function() {
  // 3. Este código só roda QUANDO o botão é clicado
  resultado.textContent = 'O botão foi clicado!';
  resultado.classList.add('box-verde');
});

    Outros eventos comuns:

    • 'click': Clique do mouse.
    • 'mouseover' / 'mouseout': Mouse entra / sai do elemento.
    • 'input': Usuário digita algo em um <input> ou <textarea>.
    • 'submit': Um formulário (<form>) é enviado.
    • 'keydown': Uma tecla é pressionada.
    • 'scroll': O usuário rola a página.

    6.2 O Objeto 'event'

    A função do callback sempre recebe um parâmetro especial, geralmente chamado de event (ou e), que contém informações sobre a ação.

    • event.target: Diz qual elemento disparou o evento.
    • event.preventDefault(): Impede o comportamento padrão. Essencial para formulários (impede a página de recarregar) e links (impede de navegar).
    const form = document.querySelector('form');

form.addEventListener('submit', function(event) {
  // Impede o formulário de recarregar a página
  event.preventDefault(); 
  
  console.log('Formulário enviado via JS!');
});


const input = document.querySelector('input');

input.addEventListener('input', function(event) {
  // event.target é o próprio <input>
  console.log('Valor atual: ', event.target.value);
});
